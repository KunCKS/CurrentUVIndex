# 紫外線即時監測站

> 歡迎您蒞臨，這份作品是以串接行政院環境保護署 OPEN DATA API 來練習 JavaScript

<https://kuncks.github.io/CurrentUVIndex/index.html>

## 簡介

此作品為 JavaScript 前端練習，主要功能有以下:

	1. 紫外線指即時監測
	2. 依地區顯示資料
	3. 關注喜歡的地區

## 使用技術&練習技術

- JavaScript(ES6)
- API
- Bootstrap4
- RWD
- SCSS
- Gulp

## 聲明

本作品內圖片、內容等，純粹為個人練習前端使用，不做任何商業用途。
